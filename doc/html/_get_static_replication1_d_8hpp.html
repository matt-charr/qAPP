Price the payoff by decomposing it into a linear combination of vanilla options and zero coupons.

Warning
The contract is eligible if it contains only one fixing with qAPI::AssetKind::EQUITY or qAPI::AssetKind::FOREIGN-EXCHANGE underlying attached.
User-provided market data is restricted to premium users only.
By default, your market will be defined according to data/my_data_file.json.
Any modification to this file will not be effective.
